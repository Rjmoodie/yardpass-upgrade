# YardPass Marketing Site - Design Guidelines

## Brand Colors

### Primary Palette
- **Primary**: `#6366F1` (Indigo)
- **Primary Dark**: `#4F46E5`
- **Primary Light**: `#818CF8`

### Secondary Palette
- **Secondary**: `#EC4899` (Pink)
- **Accent**: `#F59E0B` (Amber)
- **Success**: `#10B981` (Green)

### Neutral Palette
- **Background**: `#FFFFFF`
- **Surface**: `#F9FAFB`
- **Border**: `#E5E7EB`
- **Text Primary**: `#111827`
- **Text Secondary**: `#6B7280`

## Typography
- **Font Family**: Inter (Google Fonts)
- **Headings**: Bold (700)
- **Body**: Regular (400)
- **CTAs**: Medium (500)

## Component Styles
- **Buttons**: Rounded corners (8px), shadow on hover
- **Cards**: White background, subtle shadow, 12px radius
- **Forms**: Clean inputs, clear validation states
- **Icons**: Lucide React icons

## Spacing System
Use 8px grid: 8, 16, 24, 32, 48, 64, 80, 96

## Responsive Design
- Mobile-first approach
- Breakpoints: 640px, 768px, 1024px, 1280px
